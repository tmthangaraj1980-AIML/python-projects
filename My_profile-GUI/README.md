# 🖥️ AIML Profile GUI using Python Tkinter

## 📌 Project Overview
This project displays my personal AIML profile using a GUI built with Python Tkinter.

- Displays Name and Role
- Shows Learning Technologies (Python, SQL, AI/ML, DL, NLP, LLM, German)
